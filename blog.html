Question-1: What are the different ways to select an element in the DOM?

There are 5 ways to select an element in dom. These are: getElementById(), getElementsByClassName(), getElementsByTagName(), querySelector(), querySelectorAll()

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML = Returns the html tag and also the tag content
innerText = Returns the text-content only which is visible. it ignores hidden texts and html tags.
textContent = Returns the text content only including hidden texts and ignore html elements

Question-3: What is event delegation in the DOM?

Event delegation in DOM is a process where we add event listener to a parent element instead of adding it to each child element

Question-4: What is event bubbling in the DOM?

Event bubbling is a DOM concept. It happens when an event triggered and element and that event continues to bubble up to it's parent element and so on until it reaches the root or outermost element in the dom

Question-5: How do you create, add, and remove elements using JavaScript

To create, add and remove elements in javascript we can use methods like createElement() for creating child, appendChild() for adding child, removeChild() for removing child
